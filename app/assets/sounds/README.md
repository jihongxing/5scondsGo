# 音效文件

请在此目录下添加以下音效文件：

1. `coin.mp3` - 金币/下注音效（约1.5秒）
2. `win.mp3` - 获胜音效（可选）

## 推荐音效来源

- [Freesound](https://freesound.org/) - 搜索 "coin" 或 "coins"
- [Mixkit](https://mixkit.co/free-sound-effects/) - 免费音效
- [Zapsplat](https://www.zapsplat.com/) - 免费音效库

## 音效要求

- 格式：MP3
- 时长：1-2秒
- 大小：尽量小于100KB

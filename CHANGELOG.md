# Changelog

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-12-12

### 新增

#### 核心功能
- 实时多人押注游戏系统
- 5阶段游戏循环 (等待→倒计时→下注→游戏→结算)
- Commit-Reveal 可验证随机算法
- 自动下注与结算机制
- 完整的资金流水记录

#### 社交功能
- 观战模式 - 观看游戏不参与下注
- 房间聊天 - 实时聊天，敏感词过滤
- 表情反应 - 12种预设表情快速互动
- 好友系统 - 添加好友，查看在线状态
- 邀请功能 - 邀请链接，24小时有效

#### 用户体验
- 多语言支持 (中/英/日/韩)
- 5种房间主题皮肤
- 游戏记录查询和回放
- 钱包管理和收益统计
- 下注音效反馈

#### 运营管理
- Flutter Web 管理后台
- 用户管理
- 房间管理
- 资金审批
- 风控系统
- 告警中心
- 监控仪表盘

#### 技术特性
- WebSocket 实时通信
- Redis 余额缓存
- 乐观锁防并发冲突
- Prometheus 监控指标
- 结构化日志 (Zap)
- Docker 部署支持

### 技术栈
- 后端: Go 1.21
- 前端: Flutter 3.16
- 数据库: PostgreSQL 15
- 缓存: Redis 7
- 监控: Prometheus + Grafana
